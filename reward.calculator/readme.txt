Project Type: maven
Technologies Used: java11,h2,springboot

Instructions for running the solution
1. import the code from GitHub to your local machine IDE(eclipse/intellij/STS)
2. configure as maven project
3. modify the data set up script(located at path /src/main/resources/data.sql) if required.
4. modify the port number in application.properties if requires
5. run the project as spring boot application
6. once application is started then access the api from postman
	http://localhost:<port-number>/v1/customer/reward/<customer-id>
	Method Type : GET
